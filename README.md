# Game_wandering_king
# Rules
Rules are simple, don't get caught by robbers:<hr>
 - You have 5 arrows fire them by pressing **SPACEBAR**
 - Move by pressing **ARROWS**
 - Colect **AMMO KIT** to refill amunition
 - **AMMO KIT** change location when collected<hr>
